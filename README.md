# Color-Recognition

This project uses OpenCV in Python to recognize colors in an image. The program reads an image, detects multiple colors, and displays the name of each detected color.

# Tools and Technologies

- Python 
- OpenCV 
- NumPy
- Matplotlib
---

# Required Libraries

- opencv-python
- numpy
- matplotlib

Install them using:
```python
!pip install opencv-python numpy
%pip install matplotlib
```

# How to Run

1. Open the project in Visual Studio Code.
2. Install the required libraries.
3. Open `openCV.ipynb`.
4. Run all cells.
5. The program will display the image with the detected colors.
